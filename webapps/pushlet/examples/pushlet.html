WCQ - ICQ/MSN-like IM app This example illustrates using both publishing and subscribing. Presence is completely arranged by the clients who publish each other regular 'presence' events. There is no state on the server. When a user is entering/leaving the list a short sound is played. You can thus minimize the window and hear when users are entering/leaving. By clicking on a user you may send him/her with a short message. run wcq.html
