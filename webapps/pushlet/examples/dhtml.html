DHTML Examples

These show some examples of the power of Dynamic HTML. DHTML refers to a combination of JavaScript, Cascading Style Sheets (CSS) and the Document Object Model (DOM) used within type 4+ browsers. So called DHTML "Layers" are an important feature when making content dynamic. A Layer is a sort of mini document with a variable position on the screen. It can be moved around and its content can be dynamically refreshed.
